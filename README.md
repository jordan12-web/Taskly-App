Taskly
A simple to-do app built with Flutter and Hive for local task tracking.
 Features

- Add, edit, and delete tasks
- Mark tasks as completed
- Stores data locally using Hive
- Clean and minimal UI

  
 Tech Stack
- Flutter & Dart
- Hive for local storage
